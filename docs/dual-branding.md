[Testing](#testing)\n\n## Brand Configuration System\n\nThe dual branding system uses a centralized configuration approach:\n\n1. Brand detection happens through:\n   - Android: Product flavors (`ezyBiz` and `callConcierge`)\n   - iOS: Build schemes/targets\n   - React Native: Environment variables and build-time variables\n\n2. The configuration is accessible in JavaScript through:\n   - `src/brand/config.js` - Contains brand-specific settings\n   - `src/brand/BrandProvider.js` - React Context provider\n   - `src/brand/useBrand.js` - Custom hook to access brand configuration\n\n## Adding Brand-Specific Assets\n\n### Android Assets\nPlace brand-specific assets in the respective flavor directories:\n```\nandroid/app/src/ezyBiz/res/\nandroid/app/src/callConcierge/res/\n```\n\nFor example, for launcher icons:\n```\nandroid/app/src/ezyBiz/res/mipmap-mdpi/ic_launcher.png\nandroid/app/src/callConcierge/res/mipmap-mdpi/ic_launcher.png\n```\n\n### iOS Assets\nPlace brand-specific assets in the respective Xcode asset catalogs:\n```\nios/EzyBiz/Images.xcassets/\nios/CallConcierge/Images.xcassets/\n```\n\n### React Native Assets\nFor cross-platform assets used in JavaScript:\n```\nsrc/assets/ezyBiz/\nsrc/assets/callConcierge/\n```\n\nImport assets based on the current brand:\n```javascript\nimport { useBrand } from './src/brand/useBrand';\n\nconst MyComponent = () => {\n  const { brandConfig } = useBrand();\n  const logo = brandConfig.assets.logo;\n  // Use logo in your component\n};\n```\n\n## Brand-Specific Configuration\n\nBrand configurations are defined in `src/brand/config.js`:\n\n```javascript\nconst BRANDS = {\n  EZY_BIZ: 'ezyBiz',\n  CALL_CONCIERGE: 'callConcierge'\n};\n\nconst BRAND_CONFIG = {\n  [BRANDS.EZY_BIZ]: {\n    appName: 'EzyBiz',\n    primaryColor: '#007AFF',\n    apiEndpoint: 'https://api.ezybiz.com',\n    assets: {\n      logo: require('../assets/ezyBiz/logo.png'),\n      // other assets\n    }\n  },\n  [BRANDS.CALL_CONCIERGE]: {\n    appName: 'CallConcierge',\n    primaryColor: '#FF6B35',\n    apiEndpoint: 'https://api.callconcierge.com',\n    assets: {\n      logo: require('../assets/callConcierge/logo.png'),\n      // other assets\n    }\n  }\n};\n\nexport { BRANDS, BRAND_CONFIG };\n```\n\nTo add new brand-specific properties:\n1. Add the property to both brand configurations in `src/brand/config.js`\n2. Use the property in your components via `useBrand()` hook\n\nExample:\n```javascript\n// In src/brand/config.js\nconst BRAND_CONFIG = {\n  [BRANDS.EZY_BIZ]: {\n    // existing properties...\n    newProperty: 'ezyBizValue'\n  },\n  [BRANDS.CALL_CONCIERGE]: {\n    // existing properties...\n    newProperty: 'callConciergeValue'\n  }\n};\n\n// In your component\nimport { useBrand } from './src/brand/useBrand';\n\nconst MyComponent = () => {\n  const { brandConfig } = useBrand();\n  console.log(brandConfig.newProperty); // Will be different for each brand\n};\n```\n\n## Build Commands\n\nThe following npm scripts are available for building and running the apps:\n\n### Android\n- `npm run build:ezyBiz` - Builds EzyBiz Android release\n- `npm run build:callConcierge` - Builds CallConcierge Android release\n- `npm run run:ezyBiz` - Runs EzyBiz on Android device/emulator\n- `npm run run:callConcierge` - Runs CallConcierge on Android device/emulator\n\n### iOS\n- `npm run ios:ezyBiz` - Runs EzyBiz on iOS simulator/device\n- `npm run ios:callConcierge` - Runs CallConcierge on iOS simulator/device\n\n## Switching Between Brands\n\nDuring development, you can switch between brands:\n\n1. **For Android development**:\n   ```bash\n   # To run EzyBiz\n   npm run run:ezyBiz\n   \n   # To run CallConcierge\n   npm run run:callConcierge\n   ```\n\n2. **For iOS development**:\n   ```bash\n   # To run EzyBiz\n   npm run ios:ezyBiz\n   \n   # To run CallConcierge\n   npm run ios:callConcierge\n   ```\n\n3. **For building release versions**:\n   ```bash\n   # To build EzyBiz\n   npm run build:ezyBiz\n   \n   # To build CallConcierge\n   npm run build:callConcierge\n   ```\n\n## Testing\n\nTo test both brand builds:\n1. Ensure you have emulators/simulators set up for both Android and iOS\n2. Run the brand-specific test commands:\n   - Android: `npm run run:ezyBiz` and `npm run run:callConcierge`\n   - iOS: `npm run ios:ezyBiz` and `npm run ios:callConcierge`\n\nEach brand should display its unique styling, assets, and configuration as defined in the brand configuration files."]
